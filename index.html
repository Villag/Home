What is Villag?

Villag is a vision set forth by Patrick Daly with the loosely defined goal of making things that improve Denton, TX and will consenquently be able to help improve other communities.

Think of Villag as a single reference point for open-source projects going on in Denton. Villag helps to connect people to projects they can benefit from or contribute to.

What's here today is simply a vision. Villag is not an organization (though it could be).

Villag Projects

★ CreateDenton On Github

The goal of CreateDenton is to forge real connections among Denton, Texas' creative industry to help create work in Denton, create beauty in Denton, create tools to solve problems in Denton ... to create a better Denton.

★ VerbDenton On Github

VerbDenton compiles the awesome things going on Denton, TX and makes them easy to find.

★ VoteDenton On Github

VoteDenton is an effort to help Dentonites find which district they live in, find out if they're registered to vote, find information about candidates, and ultimately help people vote!
